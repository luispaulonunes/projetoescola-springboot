# Sistema de Gerenciamento de Usuários

Projeto desenvolvido para a disciplina Imersão Profissional: Desenvolvimento de Aplicação Web.

## Objetivo

O sistema permite o gerenciamento de usuários através de uma API REST desenvolvida com Spring Boot, utilizando autenticação JWT e persistência de dados em MySQL.

## Tecnologias Utilizadas

- Java 21
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- Maven
- MySQL
- JavaScript
- Git e GitHub
